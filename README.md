📄 Инструкция по запуску проекта "Transport Routes App"
Данный проект упакован в Docker для обеспечения идентичности окружения.

1. Предварительные требования
На компьютере должен быть установлен Docker Desktop (включает Docker Compose).
Скачать Docker Desktop: https://www.docker.com/products/docker-desktop/
(Вам не нужно устанавливать Python или зависимости)

2. Запуск приложения
Распакуйте архив с проектом в рабочую директорию.
В рабочей директории откройте терминал (CMD / PowerShell / Terminal).
Выполните команду для сборки и запуска:
    docker-compose up --build
Дождитесь завершения сборки. 
Приложение будет доступно по адресу: http://localhost:5000

3. Учетные данные для проверки
Админка: http://localhost:5000/admin
Логин: admin
Пароль: admin_iset

4. Основные команды
Остановка проекта: Ctrl + C в терминале.
Запуск в фоновом режиме: docker-compose up -d.
Просмотр логов: docker-compose logs -f.
